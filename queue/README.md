# Queue

David-facing intake lanes. Drop things here when the desired outcome is known but the exact workflow change is not.

```text
01_PROMPTS/           prompts that should evolve the system
02_FILES/             loose files needing intake
03_WORKFLOW_IDEAS/    rough process ideas
04_CORRECTIONS/       explicit fixes and overrides
05_REVIEW_DECISIONS/  approvals/rejections for pending plans
```

Nothing in queue is source truth until a workflow copies it into a packet, logs provenance, and receives approval.
