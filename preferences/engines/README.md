# Preference Engines

Configs and adapters for the preference stack. Learned artifacts live in NAS model slots; GitHub stores only repo-safe configs and contracts.

| Engine | Role |
|---|---|
| River | hot online learning |
| PPK | portable identity weights |
| Implicit | co-occurrence recommendation |
| LightFM | hybrid metadata + behavior recommendation |
| Markovify | lightweight voice/text prediction |
