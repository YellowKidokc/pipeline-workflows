# Trinity Canon Station

Builds the canon index for resurrection, Maxwell/Trinity, and Lean isomorphism material.

## Run

```powershell
python Backside\stations\trinity-canon.station\station.py --out X:\EXPORTS\canon-index\trinity
```

## Output

- `canon-index.json`
- `canon-index.md`

This station is deterministic. It does not write to Postgres.
