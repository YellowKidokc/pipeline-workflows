# Operators Canon Station

Builds the canon index for Grace, Justice/Mercy, and related operator/data material.

## Run

```powershell
python Backside\stations\operators-canon.station\station.py --out X:\EXPORTS\canon-index\operators
```

## Output

- `canon-index.json`
- `canon-index.md`

This station is deterministic. It does not write to Postgres.
