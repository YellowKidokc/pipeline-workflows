# Master Equation Canon Station

Builds the canon index for the Master Equation, formal coherence layer, unified test stack, and axiom derivation chain.

## Run

```powershell
python Backside\stations\master-equation-canon.station\station.py --out X:\EXPORTS\canon-index\master-equation
```

## Output

- `canon-index.json`
- `canon-index.md`

This station is deterministic. It does not write to Postgres.
