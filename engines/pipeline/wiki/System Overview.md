# FAP System Overview

Folder Automations & Pipelines is the station-based workflow layer for BIL/PIL.

The operating idea:

```text
Folders are stations.
Pipelines are rails.
Manifests are passports.
Signals are feedback.
LLM hubs are thinking checkpoints.
Wiki pages explain the machine.
```

The system should be rigorous enough that every artifact has a visible path,
visible status, and visible reason for advancement or rejection.
